

Welcome to Scala Instruments Library documentation!
=========================================

Required Scala version: 2.12.x

To use this library, add the following dependency to your POM: 

.. code-block:: xml
    :linenos:
    
    <dependency>
        <groupId>org.odfi.instruments</groupId>
        <artifactId>instruments-core</artifactId>
        <version>0.0.1-SNAPSHOT</version>
    </dependency>
